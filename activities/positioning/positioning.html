1. It's the default way elements are displayed on a page, block elements stack vertically, and inline elements go left to right

2. Float moves an element to teh left or right, allowing other cocntent to wrap around it like tect around an image.

3. To stop elements after a floated element from wrapping around it, clear resets the normal layout flow.

4. Relative: moves the element relative to its normal position

Absolute: positions it exactly within its parent

Fixed: Sticks the element to a specific spot on teh screen. It doesnt move when we scroll
